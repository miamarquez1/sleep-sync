# SleepSync

SleepSync is a beginner-friendly sleep tracking web app where users log sleep times, track streaks, and visualize sleep habits. This project is designed as a midterm web app using vanilla HTML, CSS, and JavaScript, with future Firebase authentication and database integration.

## Planned Features
1. Sleep log form with bedtime and wake time inputs
2. Streak tracking for consistent sleep habits
3. Visual sleep insights (weekly/monthly trends)
4. Personalized reminders and sleep goals
5. Firebase authentication and cloud storage

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Netlify (deployment)
- Firebase (planned)

## Setup Instructions
1. Clone the repository.
2. Open `sleep-sync/index.html` in your browser to view the landing page.
3. Edit files in `sleep-sync/` to customize styling, content, and behavior.
4. Deploy to Netlify by connecting the GitHub repository and setting the publish directory to `sleep-sync/`.

## Project Structure
- `index.html` — landing page
- `css/styles.css` — main styles
- `js/app.js` — UI logic
- `js/storage.js` — local storage layer (placeholder for Firebase later)
- `pages/` — login, signup, and dashboard pages
